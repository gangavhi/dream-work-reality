# iOS Scaffold (SwiftUI + XcodeGen)

This directory contains a reproducible iOS scaffold generated by XcodeGen.

## Prerequisites

- Xcode 15+
- Xcode command line tools
- [XcodeGen](https://github.com/yonaskolb/XcodeGen)
- Rust toolchain (`rustup`) for the shared core build script

Install XcodeGen if needed:

```bash
brew install xcodegen
```

Install Rust target for Apple Silicon simulator if needed:

```bash
rustup target add aarch64-apple-ios-sim
```

## Generate the Xcode project

```bash
cd apps/ios
xcodegen generate
```

This creates `DreamWorkApp.xcodeproj` from `project.yml`.

## Run tests in simulator via xcodebuild

```bash
cd apps/ios
xcodebuild \
  -project DreamWorkApp.xcodeproj \
  -scheme DreamWorkApp \
  -destination 'platform=iOS Simulator,name=iPhone 15,OS=latest' \
  test
```
